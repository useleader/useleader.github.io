研究经历一：工业控制程序自动化逆向分析方法

针对工业控制程序格式封闭、指令体系私有、人工逆向分析依赖专家经验且成本高的问题，我设计了一种面向工业控制程序的自动化逆向分析方法，旨在提升工业控制系统安全分析的规模化与可复用能力。

研究依托： 本研究基于国家自然科学基金青年科学基金项目（C类）展开，面向电力与工业控制系统中广泛存在的 PLC 程序与私有控制逻辑，围绕其可解析性、可分析性与安全评估需求，开展自动化逆向分析方法研究。

研究内容： 我围绕工业控制程序文件的结构解析与控制逻辑恢复开展系统研究，通过对同源程序文件的格式归纳与字段抽象，实现子程序段的自动分割；进一步结合反汇编、符号执行与函数识别技术，对工业控制程序中的静态库函数、动态库函数、用户自定义函数及设备 I/O 操作函数进行自动识别与分类，并重建函数调用关系与控制流结构。最终，基于函数指纹与调用图构建机制，实现逆向结果的结构化表达与可视化展示，支持不同厂商、不同型号 PLC 程序的批量化逆向分析。

主要贡献： 构建了一套面向工业控制程序的自动化逆向分析流程，显著降低了对人工经验与手工分析的依赖，适用于多品牌、多型号 PLC 程序；提出基于函数指纹与调用关系图的控制逻辑恢复方法，为后续漏洞挖掘、攻击面分析与安全评估提供关键中间表示；实现逆向结果的结构化与可视化输出，提升工业控制系统安全分析的效率与可复用性。

技术沉淀： 通过该研究，系统积累了针对私有二进制格式的文件结构归纳与自动解析经验，深入掌握了反汇编、符号执行与函数识别技术在工业控制程序中的协同应用方法；加深了对 PLC 程序执行模型、函数调用机制与 I/O 映射关系的理解，并具备将逆向分析结果转化为可视化安全分析工件的研究与工程能力。

产出成果： 申请并获授权国家发明专利一项《一种工业控制程序自动化逆向分析方法》（申请号：202311329186.0，已授权，排名第二），相关技术方案已形成完整权利要求与可实施技术路线。

研究经历二：基于多智能体协作的代码指令调优方法研究

针对单一大语言模型在复杂代码生成与指令理解任务中易出现推理偏置、上下文利用不足以及结果稳定性有限等问题，我围绕多智能体协作机制开展了代码指令调优方法研究，并设计实现了 xCoder 系统。

研究依托： 本研究基于第34届冯如杯竞赛主赛道与2024年“挑战杯”中国大学生创业计划竞赛展开，面向大语言模型在真实开发与安全分析场景中的工程化应用需求，探索多智能体协作范式在代码生成与指令优化任务中的可行性与实际效果。

研究内容： 我提出并实现了 xCoder——一个基于多智能体协作的代码指令调优工具。该系统通过构建由“需求理解智能体、代码生成智能体、审查与优化智能体”组成的协作架构，将复杂代码生成任务拆解为多阶段、可交互的子任务；各智能体围绕统一目标在共享上下文中进行角色分工与结果博弈，通过多轮反馈与交叉修正，逐步消解单模型生成过程中存在的语义偏差与逻辑缺陷，从而实现对用户代码指令的自动调优与生成质量提升。

主要贡献： 设计了一种面向代码生成任务的多智能体协作机制，将指令理解、代码生成与质量审查解耦为可组合的智能体角色，提升系统整体鲁棒性；提出基于多轮协商与反馈的指令调优流程，有效缓解大模型在复杂需求下的幻觉问题与生成不稳定性；实现具备实际可用性的工程化工具，在多类代码生成与优化场景中验证了该方法在生成正确性、可读性与任务完成度方面的优势。

技术沉淀： 通过该研究，系统掌握了多智能体系统的任务拆解与协作设计方法，深化了对大语言模型在代码理解、生成与审查等不同角色下能力边界的认识；积累了多智能体上下文管理、交互协议设计以及面向真实用户需求的 LLM 工程化落地经验，为后续开展大模型安全、自动化代码分析与复杂任务推理研究奠定了坚实基础。

产出成果： 完成作品 xCoder——基于多智能体协作的代码指令调优工具，并作为核心成果参加第34届冯如杯竞赛主赛道及2024年“挑战杯”中国大学生创业计划竞赛。

研究经历三：基于大语言模型的通用模糊测试框架优化研究

针对现有通用模糊测试工具在测试样本生成依赖人工规则、跨程序泛化能力弱以及大语言模型直接参与模糊测试时存在成本高、稳定性不足等问题，我主导设计并实现了 FuzzGen，系统性探索大语言模型赋能通用模糊测试的新范式。

研究依托： 本研究基于国家信息安全专项“基于大语言模型的开源模糊测试框架优化技术与应用”展开，围绕开源模糊测试工具的智能化升级与工程化落地需求，面向真实软件系统的漏洞挖掘场景开展方法设计与系统实现。

研究内容： 我提出了 FuzzGen，一个面向多语言、多程序的 LLM 驱动通用模糊测试框架。该框架以“程序语义理解—输入生成—反馈驱动优化”为核心流程，引入大语言模型对目标程序进行自动语义分析，抽取隐式输入约束、关键路径及潜在错误触发模式；在此基础上，构建可迁移的测试输入生成策略，实现对不同程序和语言的统一支持。同时，FuzzGen 通过轻量化提示模板与执行反馈闭环机制，将运行时覆盖率与异常信息映射为对 LLM 的结构化反馈，在不依赖复杂人工规则的前提下持续优化生成质量，有效控制推理开销。

主要贡献： 构建一种面向通用程序的 LLM 增强模糊测试框架，突破传统模糊测试对手工输入建模和专家经验的依赖；提出基于程序语义理解的输入生成与反馈驱动优化方法，显著提升模糊测试在不同程序间的泛化能力与稳定性；在多种真实世界开源项目上的实验结果表明，FuzzGen 在代码覆盖率、崩溃触发效率及漏洞发现数量方面均优于主流模糊测试工具。

技术沉淀： 通过该研究，系统积累了大语言模型在程序分析与软件测试场景下的提示工程设计经验，掌握了 LLM 与灰盒模糊测试框架的深度融合方法；深化了对通用程序输入空间建模、执行反馈抽象及跨程序测试策略迁移问题的理解，并具备从算法设计到原型系统实现与实验评估的完整研究与工程能力。

产出成果： 完成 FuzzGen 原型系统并在国家信息安全专项中通过阶段性验收，相关研究成果已整理为一整套模糊测试系统，为后续高水平漏洞挖掘奠定基础。

研究经历四：基于大语言模型的字节级模糊测试方法

针对传统模糊测试在字节级变异阶段存在变异盲目性强、低收益突变比例高、难以精准触达关键执行路径等问题，我开展了基于大语言模型的字节级模糊测试方法研究，探索以模型驱动方式替代传统随机或启发式字节变异策略。

研究依托： 本研究基于国家信息安全专项“基于大语言模型的开源模糊测试框架优化技术与应用”展开，围绕通用模糊测试框架在低层字节变异阶段智能化程度不足的问题，面向真实软件系统与开源项目的漏洞挖掘需求，设计可工程化落地的字节级智能模糊测试方法。

研究内容： 我将字节级突变过程形式化为统一的序列学习任务，以原始输入字节序列作为模型输入，以突变位置与突变策略作为联合输出，利用大语言模型对“在哪里变、如何变”进行协同预测。同时，引入真实模糊测试历史作为监督信号，通过监督微调与反馈式序列更新机制，使模型能够在字节粒度上识别高价值变异区域，并根据执行反馈持续自优化突变策略，从而显著提升新路径触达与异常触发的效率。

主要贡献： 首次在字节级模糊测试中系统引入大语言模型对突变位置与突变策略的联合预测机制，将低层随机变异转化为模型引导的精细化决策过程；构建基于真实模糊测试历史的监督训练数据集，增强模型在复杂二进制格式与多类型输入场景下的泛化能力；显著降低低收益字节变异比例，在相同计算资源约束下有效提升漏洞触发概率与测试效率。

技术沉淀： 通过该研究，系统积累了将模糊测试低层操作抽象为序列学习问题的建模经验，掌握了基于真实模糊历史构建监督微调数据集的方法；深入理解了字节级输入空间、变异策略与程序执行路径之间的内在关联，并具备将大语言模型与传统灰盒模糊测试反馈机制深度融合、实现持续自优化测试流程的研究与工程能力。

产出成果： 申请国家发明专利一项《一种基于大语言模型的字节级模糊测试方法》（已受理，申请号：202511800067.8，排名第二），相关技术方案已形成完整专利说明书与实施例。

研究经历五：针对协议模糊测试的优化研究

针对现有状态感知协议模糊测试在复杂协议语义理解不足、状态空间探索效率低以及大语言模型推理成本过高等问题，我系统性地开展了协议模糊测试的优化研究，探索大语言模型在复杂协议测试场景下的高效决策与工程化应用路径。

研究依托： 本研究基于国家自然科学基金面上项目“工业物联网安全映射机理与防护方法研究”展开，面向工业物联网与网络协议栈中广泛存在的状态复杂、语义耦合度高的通信协议，实现对协议实现漏洞的高效发现与系统性安全评估。

研究内容： 我提出了 HieFuzz，一个分层式的大语言模型增强智能模糊测试框架。HieFuzz 创新性地引入三层递进式认知架构：语法理解层用于保障协议消息结构的合法性与完整性；语义推理层聚焦协议状态迁移与复杂逻辑依赖关系建模；策略优化层则基于历史执行反馈与风险评估结果动态调制突变策略。该分层设计有效解耦协议模糊测试中的多重决策任务，并通过协议感知的多层缓存机制显著降低大语言模型的推理开销。

主要贡献： 提出一种分层协同的大语言模型驱动协议模糊测试架构，系统性缓解现有 LLM 模糊测试方法中的决策瓶颈与上下文混淆问题；设计面向协议语义的缓存与反馈机制，在保证测试深度与状态探索充分性的同时显著降低计算与推理成本；在多种真实协议实现上的实验结果表明，该方法在状态覆盖率、代码覆盖率及漏洞发现能力方面均取得显著提升。

技术沉淀： 通过该研究，系统积累了将复杂协议模糊测试任务进行分层建模与决策解耦的设计经验，深化了对协议语法、语义与状态演化之间内在关系的理解；掌握了大语言模型在高频决策场景下的推理成本控制与缓存优化方法，并具备将 LLM 与状态感知模糊测试反馈机制深度融合、实现可扩展协议测试系统的研究与工程能力。

产出成果： 以第二作者身份撰写论文《HieFuzz: A Hierarchical LLM Framework for Stateful Protocol Fuzzing》，目前IEEE Transactions on Big Data 期刊在投。

研究经历六：固件与协议漏洞挖掘

在理论研究之外，我长期参与真实系统环境下的漏洞挖掘工作，面向网络设备与工业控制设备中的固件与协议实现，开展系统性的安全分析与漏洞发现研究。

研究依托： 本研究基于多项开源安全奖励计划，并与前述国家级科研项目在方法与技术路线上保持协同推进。在持续研发与实验过程中，将协议模糊测试、固件逆向分析及工业控制程序分析等研究成果应用于真实设备环境，形成面向工程实践的长期漏洞挖掘工作。

研究内容： 我围绕路由器、工业控制设备及 PLC 系统的实际运行环境，综合运用协议语义分析、固件逆向工程与模糊测试技术，对设备通信协议、配置解析逻辑及控制程序实现中的安全缺陷进行系统挖掘；通过构建可复现的测试环境与漏洞触发路径，对发现的问题进行验证、定位与影响分析。

主要贡献： 发现并验证多项真实网络设备与工业控制系统中的安全漏洞，涵盖命令执行与工控系统逻辑缺陷等典型问题；将前期提出的自动化逆向分析与智能模糊测试方法成功应用于真实设备环境，验证相关研究方法在工程实践中的有效性与可迁移性；为相关设备厂商与行业漏洞库提供可复用的技术分析与修复依据。

技术沉淀： 通过长期的漏洞挖掘实践，系统积累了面向真实设备环境的漏洞发现与复现经验，深化了对网络协议实现缺陷、固件解析逻辑与工业控制程序运行机制的理解；形成了从漏洞假设构建、触发条件设计到影响评估与披露的完整技术流程，并具备将科研方法快速转化为工程化漏洞挖掘能力的实践经验。

产出成果： 联合申报并公开多项 CNVD 漏洞（包括 CNVD-2025-07982、CNVD-2025-01950、CNVD-2025-01941 等），涉及小米 AX1800、友讯 DIR-830L A1 以及信捷 XD / XL 系列 PLC 等网络与工业控制设备。
