# agentflow 学习笔记

## 项目概览

**类型**: Python 编排框架  
**用途**: 在依赖图中协调多个 AI Agent (Codex, Claude, Kimi) 执行复杂任务流水线  
**技术栈**: Python + FastAPI (Web UI)

## 核心功能

### Fanout (并行扇出)
- 将节点发散为 N 个并行副本
- 支持按列表、整数或笛卡尔积方式分发

### 迭代循环
- 通过 `on_failure` 实现写→review→修复循环
- 直到满足成功条件

### 远程执行
- SSH/EC2/ECS 零配置远程运行 Agent

### 调优 Agent 演化
- 从运行轨迹中训练专用调优 Agent

## 架构分析

```
orchestrator.py  - 执行引擎
dsl.py          - Graph/fanout/merge 定义
specs.py        - 成功标准
graph_optimizer.py - 图优化
```

### 多后端支持
通过 `pi` 统一路由至:
- OpenAI / Anthropic / Groq / Cerebras
- 本地 LMStudio / Ollama
