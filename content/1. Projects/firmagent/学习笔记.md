# firmagent 学习笔记

## 项目概览

**类型**: 学术研究论文复现项目  
**来源**: NDSS 2026  
**目标**: IoT固件安全研究（路由器 + PLC）

## 核心架构

### Multi-Agent 协作
- Fuzzing Agent + Analysis Agent + Coordinator
- ReAct 推理模式

### Coverage-guided Fuzzing
- AFL++ (通用)
- AFLNet (网络协议)

### 目标设备
- Home Router (网络协议)
- PLC (工控协议 S7comm/Modbus)

## 技术栈

| 组件 | 工具 |
|------|------|
| Fuzzing | AFL++, AFLNet, QEMU |
| 固件分析 | FirmAE, binwalk, Ghidra |
| LLM Agent | Anthropic Claude API, LangChain/LangGraph |
| PLC 通信 | Snap7 (西门子 S7), pymodbus |
