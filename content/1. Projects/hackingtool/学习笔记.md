# hackingtool 学习笔记

## 项目概览

**类型**: 安全工具聚合器  
**规模**: 185+ 开源工具，20 个攻击/安全测试类别  
**技术栈**: Python 3.10+, Rich CLI框架

## 主要功能

- 信息收集、云安全、DDOS、漏洞利用
- 取证、逆向工程、钓鱼攻击
- 后渗透、无线攻击、Web 攻击
- SQL/XSS 注入、隐写术、字典生成

## 架构分析

```
入口点 → HackingToolsCollection 分类菜单 → HackingTool 基类工具
```

### 核心模块
- `core.py` - 框架核心
- `tools/` - 20个分类模块
- `install.py` - 安装脚本
- `tool_manager.py` - 工具状态检测
