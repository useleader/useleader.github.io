## 课程概览

Hugging Face 官方出品的 Context Engineering 课程，从 Agent Skills、MCP 协议、插件系统、子智能体、钩子系统到从零实现 Agent Loop，系统学习如何为 AI Code Agent 构建高效上下文。

- **课时**: 每单元约 1 周，2-3 小时/周
- **形式**: 理论 + Hands-on + 测验 + 结业项目
- **证书**: Context Fundamentals Certificate（Unit 1-2）/ Context Engineering Certificate（全部 Unit）
