## 创建原始语料库
测试的输入输出分别在`/inputs`,`/outpus`
> 如何生成高质量语料库是一块问题，如果感兴趣可以研究一下，感觉大模型应该能帮助一些


### addr2line
**目的种子**：地址列表文本（每行一个地址）、若干真实可执行文件用于 `-e` 参数。

```bash
mkdir -p /inputs/seeds/addr2line
# 从系统二进制提取若干地址
objdump -d /bin/ls | awk '/^[[:xdigit:]]+:/ {gsub(":", "", $1); print "0x"$1}' | head -n 200 > /inputs/seeds/addr2line/addr_list1.txt
# 简单地址样本
printf "0x400000\n400123\nmain+0x5\n" > /inputs/seeds/addr2line/addr_misc.txt
# 用小程序生成死地址、超长行、空文件
echo "" > /inputs/seeds/addr2line/empty
python3 - <<'PY' 
open('/inputs/seeds/addr2line/long.txt','w').write('0x'+'41'*5000)
PY
# 备用可执行做 -e 参数（复制系统二进制做样本）
cp /bin/ls /inputs/seeds/addr2line/ls_bin 2>/dev/null || true
mkdir -p /outputs/addr2line
```

### ar
```bash
mkdir -p /inputs/seeds/ar
# 生成一个简单 .a
echo 'int f(){return 0;}' > /tmp/_h.c
gcc -c /tmp/_h.c -o /tmp/_h.o
ar rcs /inputs/seeds/ar/libh.a /tmp/_h.o
# 从系统复制现有静态库（如存在）
cp /usr/lib/*.a /inputs/seeds/ar/ 2>/dev/null || true
# 截断 / 损坏样本
head -c 512 /inputs/seeds/ar/libh.a > /inputs/seeds/ar/libh_trunc.a
dd if=/dev/urandom of=/inputs/seeds/ar/random.bin bs=256 count=1 2>/dev/null
mkdir -p /outputs/ar

```


### bfdtest1 / bfdtest2 （两者相似：bfd 相关测试工具）

**目的种子**：`.o`、ELF、共享库、截断/损坏 object。
```bash
mkdir -p /inputs/seeds/bfdtest1 /inputs/seeds/bfdtest2
# 基本小 object
echo 'int main(){return 0;}' > /tmp/_h.c
gcc -c /tmp/_h.c -o seeds/bfdtest1/t.o
cp seeds/bfdtest1/t.o seeds/bfdtest2/t.o
# 复制系统 elf / so
cp /bin/ls seeds/bfdtest1/ls 2>/dev/null || true
cp /usr/lib/*.so seeds/bfdtest2/ 2>/dev/null || true
# 截断样本
head -c 300 seeds/bfdtest1/t.o > seeds/bfdtest1/t_trunc.o
mkdir -p /outputs/bfdtest1 /outputs/bfdtest2
```
