## 一、先决准备（必须做）
- **快照（snapshot）与隔离网络**
    - 在开始前为 VM 建快照，任何一步失败都能回滚。
        
    - 使用 Host-only / 内部网络或断网模式，**不要**把可疑样本连到真实 internet。
        
- **环境分区**
    - Kali：静态分析、二进制/PE/脚本解析、网络模拟、流量分析、脚本化工具。
        
    - 如需动态运行 Windows 二进制，推荐另外准备一台 **隔离的 Windows VM**（供 x64dbg、Procmon、Wireshark 使用），Kali 做监控与网络模拟。
        
- **基础工具安装/更新**（Kali 已预装很多，但检查）
    - `sudo apt update && sudo apt upgrade`
        
    - 安装常用包（若缺失）：`sudo apt install radare2 ghidra yara ghidra rizin file binwalk foremost python3-pip wireshark tcpdump mitmproxy socat`
        
    - Python 库：`pip3 install pefile capstone keystone-engine lief yara-python`
