# Introducing GPT-4.1 in the API

## 核心内容

OpenAI 推出三个新模型：GPT-4.1、GPT-4.1 mini 和 GPT-4.1 nano，在编码、指令遵循和长上下文方面有重大改进。

## 主要特性

1. **性能提升**：在编码和指令遵循方面优于 GPT-4o 和 GPT-4o mini

2. **上下文窗口**：支持高达 100 万 token 的上下文，并改进了长上下文理解

3. **知识截止日期**：2024年6月

4. **nano模型**：这是 OpenAI 首个 nano 模型

## 定价

- GPT-4.1 nano 是最低成本的选项
- 这些模型提供更低的成本和延迟

## 重要性

通过专注于真实的开发者需求——从编码到指令遵循和长上下文理解——这些模型为构建智能系统和复杂的代理应用程序开辟了新的可能性。
