# New Tools for Building Agents

## 核心内容

OpenAI 发布了用于构建代理的新 API 和工具集，旨在帮助开发者和企业构建有用且可靠的代理。

## 新工具发布

1. **Responses API**：结合 Chat Completions API 的简单性和 Assistants API 的工具使用能力

2. **内置工具**：
   - Web Search
   - File Search
   - Computer Use

3. **Agents SDK**：编排单代理和多代理工作流

4. **可观察性工具**：追踪和检查代理工作流执行

## 代理的定义

代理是代表用户独立完成任务的系统，需要复杂的多步骤任务处理能力。

## 价值

这些版本引入了第一批构建模块，使开发者和企业能够更轻松地构建、部署和扩展可靠、高性能的 AI 代理。
