# ODSCAN 术语表

## 技术术语

| 英文术语 | 中文术语 | 解释 |
|----------|----------|------|
| Backdoor | 后门/特洛伊木马 | 在模型中植入的隐藏触发机制 |
| Trigger Inversion | 触发器反转 | 反向工程还原后门触发器的技术 |
| Object Detection | 目标检测 | 计算机视觉中检测图像中物体位置和类别的任务 |
| Bounding Box | 边界框 | 标注物体位置的矩形框 |
| Trojaned Model | 特洛伊模型 | 被植入后门的机器学习模型 |
| Benign Model | 良性模型 | 正常无后门的模型 |
| ROC-AUC | 接收者操作特征曲线下面积 | 评估分类模型性能的指标 |
| Mean Average Precision | 平均精度均值 | 目标检测模型的主要评估指标 |
| Attack Success Rate | 攻击成功率 | 后门攻击成功的比例 |
| SSD | 单次多框检测器 | Single Shot MultiBox Detector，实时目标检测网络 |
| F-RCNN | 快速区域卷积神经网络 | Faster R-CNN，两阶段目标检测网络 |
| Non-Maximum Suppression | 非极大值抑制 | 目标检测中去除重复检测框的后处理技术 |
| Low-Confidence Filter | 低置信度过滤器 | 过滤低置信度检测结果的步骤 |
| Trigger Specificity | 触发器特异性 | 只有精确形式的触发器才能激活后门的特性 |
| Adversarial Training | 对抗训练 | 使用对抗样本增强模型鲁棒性的训练方法 |
| Meta Classification | 元分类 | 通过分析模型权重检测后门的方法 |
| TrojAI | -- | IARPA组织的后门扫描竞赛 |
