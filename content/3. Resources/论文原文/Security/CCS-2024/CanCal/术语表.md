# CanCal: 基于实时日历分析的勒索软件检测

## 技术术语

| 英文术语 | 中文术语 | 解释说明 |
| --- | --- | --- |
| CanCal | Calendar-based Ransomware Detection | 基于实时日历分析的勒索软件检测系统 |
| Ransomware | 勒索软件 | 加密受害者文件并索要赎金的恶意软件 |
| Detection | 检测 | 识别恶意软件存在的过程 |
| Real-time | 实时 | 系统在事件发生时立即进行处理 |
| Calendar Analysis | 日历分析 | 观察用户日历行为模式来检测异常 |
| Encryption | 加密 | 将数据转换为密文以防止未授权访问 |
| File Access Pattern | 文件访问模式 | 程序访问文件的时间和方式规律 |
| Behavioral Analysis | 行为分析 | 基于程序行为而非特征码来检测恶意软件 |
| False Positive | 误报 | 将正常程序错误识别为恶意软件 |
| False Negative | 漏报 | 未能检测到实际存在的恶意软件 |
