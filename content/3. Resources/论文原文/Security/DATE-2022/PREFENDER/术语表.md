# PREFENDER: 基于缓存侧信道的预取攻击防御

## 技术术语

| 英文术语 | 中文术语 | 解释说明 |
| --- | --- | --- |
| PREFENDER | Prefetch-based Defender | 基于预取的防御者，一种利用预取器防御缓存侧信道攻击的系统 |
| Cache Side-channel | 缓存侧信道 | 利用缓存访问时间差异获取敏感信息的攻击方法 |
| Prefetch Attack | 预取攻击 | 通过操纵预取器进行信息泄露的攻击 |
| Cache Timing Attack | 缓存时序攻击 | 测量缓存访问时间来判断数据是否被加载的攻击 |
| Prime+Probe | Prime+Probe | 先填充缓存再探测访问模式的侧信道技术 |
| Flush+Reload | Flush+Reload | 通过刷新共享内存并检测加载时间的侧信道技术 |
| Defense Mechanism | 防御机制 | 防止或阻止攻击的安全措施 |
| Security | 安全性 | 系统保护数据免受未授权访问的能力 |
