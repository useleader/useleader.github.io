# PreFence: 预取侧信道防御

## 技术术语

| 英文术语 | 中文术语 | 解释说明 |
| --- | --- | --- |
| PreFence | Prefetch-based Defense | 基于预取的防御系统，防御预取侧信道攻击 |
| Prefetch Side-channel | 预取侧信道 | 利用硬件预取器进行信息泄露的侧信道攻击 |
| Side-channel Attack | 侧信道攻击 | 利用系统实现中的物理效应获取敏感信息 |
| Defense | 防御 | 阻止或减轻攻击的安全措施 |
| Hardware Prefetcher | 硬件预取器 | 自动预测并预加载数据到缓存的硬件组件 |
| Cache Attack | 缓存攻击 | 通过缓存时序差异获取敏感信息的攻击 |
| Mitigation | 缓解 | 减少攻击影响的技术 |
