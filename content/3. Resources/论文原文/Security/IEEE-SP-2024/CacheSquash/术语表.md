# CacheSquash: 推测感知的安全缓存

## 技术术语

| 英文术语 | 中文术语 | 解释说明 |
| --- | --- | --- |
| CacheSquash | 推测感知缓存 | 一种防御推测执行攻击的缓存设计 |
| Speculation-Aware | 推测感知 | 能够识别和防御推测执行攻击的设计 |
| Speculative Execution | 推测执行 | 处理器提前执行尚未确认的指令 |
| Cache Attack | 缓存攻击 | 利用缓存时序差异进行的侧信道攻击 |
| Side-channel | 侧信道 | 获取敏感信息的隐蔽渠道 |
| Security | 安全性 | 系统保护数据的能力 |
| Performance | 性能 | 系统执行效率 |
