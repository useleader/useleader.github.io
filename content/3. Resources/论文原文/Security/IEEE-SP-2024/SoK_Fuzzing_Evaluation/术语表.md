# SoK: Fuzzing Evaluation 系统化研究

## 技术术语

| 英文术语 | 中文术语 | 解释说明 |
| --- | --- | --- |
| Fuzzing | 模糊测试 | 通过输入随机数据来发现程序漏洞的测试技术 |
| Evaluation | 评估 | 衡量和比较技术有效性的过程 |
| Benchmark | 基准测试 | 用于比较性能的标准化测试 |
| Bug Discovery | 漏洞发现 | 找到软件中的安全缺陷 |
| Effectiveness | 有效性 | 技术达成目标的程度 |
| Coverage | 覆盖率 | 测试覆盖的代码比例 |
