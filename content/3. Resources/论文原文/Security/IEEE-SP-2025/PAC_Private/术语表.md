# PAC-Private: Provable Privacy via Anisotropic Noise 术语表

## 技术术语

| 英文术语 | 中文术语 | 上下文/解释 |
|----------|----------|------------|
| PAC Privacy | 概率近似正确隐私 | Probably Approximately Correct Privacy，统计推断角度的隐私框架 |
| Differential Privacy | 差分隐私 | 经典的隐私定义 |
| Maximal Leakage | 最大泄漏 | 基于信息论的隐私度量 |
| Membership Inference Attack | 成员推断攻击 | 判断数据是否参与训练的推理攻击 |
| Anisotropic Noise | 各向异性噪声 | 跨输出维度变化的噪声 |
| Isotropic Noise | 各向同性噪声 | 各方向相同的噪声 |
| Black-box Algorithm | 黑盒算法 | 只能通过输入输出访问的算法 |
| Mutual Information | 互信息 | 隐私度量的信息论基础 |
| KL-divergence | KL散度 | 概率分布差异度量 |
| Stability | 稳定性 | 算法对输入变化的敏感度 |
| l2-norm | L2范数 | 向量长度度量 |
| Singular Value Decomposition | 奇异值分解 | 矩阵分解方法 |
| Covariance Matrix | 协方差矩阵 | 随机变量协同变化矩阵 |
| K-Means | K均值 | 经典聚类算法 |
| SVM | 支持向量机 | Support Vector Machine |
| PCA | 主成分分析 | Principal Component Analysis |
| Random Forest | 随机森林 | 集成学习方法 |
| Subsampling Rate | 子采样率 | 用于隐私化的数据比例 |
| Posterior Success Rate | 后验成功率 | 攻击者观察输出后的成功概率 |
| Prior Success Rate | 先验成功率 | 攻击者观察输出前的成功概率 |
| DP-SGD | 差分隐私随机梯度下降 | Differentially Private Stochastic Gradient Descent |
| PATE | 教师模型集成聚合 | Private Aggregation of Teacher Ensembles |
