# Fusion: Over-Unlearning Attacks on Machine Unlearning 术语表

## 技术术语

| 英文术语 | 中文术语 | 上下文/解释 |
|----------|----------|------------|
| Machine Unlearning | 机器遗忘 | 从模型中移除数据影响的技术 |
| MLaaS | 机器学习即服务 | Machine Learning as a Service |
| Over-Unlearning | 过度遗忘 | 遗忘超过必要范围的攻击 |
| Right to be Forgotten | 被遗忘权 | GDPR等法规赋予的权利 |
| Data Poisoning | 数据中毒攻击 | 在训练过程中注入恶意数据 |
| Retraining | 重训练 | 从头开始重新训练模型 |
| Exact Unlearning | 精确遗忘 | 保证完全移除数据影响 |
| Approximate Unlearning | 近似遗忘 | 通过参数修改近似重训练效果 |
| SISA | 顺序独立重训练 | Slicing-Independent Subset Aggregation |
| Gradient-based Unlearning | 基于梯度的遗忘 | 通过梯度更新移除影响 |
| Influence Function | 影响函数 | 衡量数据点对模型影响的方法 |
| Fine-tuning | 微调 | 在已训练模型上继续训练 |
| Unlearning API | 遗忘API | 提供遗忘服务的接口 |
| Black-box Access | 黑盒访问 | 只通过API访问模型 |
| Model Utility | 模型效用 | 模型正常任务性能 |
| Unlearning Efficacy | 遗忘效果 | 成功移除数据影响的程度 |
| Decision Boundary | 决策边界 | 分类模型的分界面 |
| Sample-level Manipulation | 样本级操作 | 对特定样本的精细修改 |
| Pixel-wise Manipulation | 像素级操作 | 对图像像素的修改 |
