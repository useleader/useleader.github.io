# Bootloader Memory Safety Analysis 术语表

## 技术术语

| 英文术语 | 中文术语 | 上下文/解释 |
|----------|----------|------------|
| Bootloader | 引导加载程序 | 系统启动时初始化硬件加载OS的程序 |
| Secure Boot | 安全启动 | 验证下一组件数字签名的机制 |
| BIOS | 基本输入输出系统 | 传统固件接口 |
| UEFI | 统一可扩展固件接口 | BIOS的现代替代方案 |
| Memory Safety | 内存安全 | 防止内存相关漏洞 |
| Fuzzing | 模糊测试 | 自动漏洞发现方法 |
| GRUB | - | Linux标准引导程序 |
| Das U-Boot | - | 嵌入式引导程序 |
| barebox | - | 嵌入式引导程序 |
| Peripheral | 外围设备 | 存储、网络、控制台等 |
| USB | 通用串行总线 | Universal Serial Bus |
| PXE | 预启动执行环境 | 网络启动协议 |
| GPT | GUID分区表 | 全局唯一标识符分区表 |
| MBR | 主引导记录 | Master Boot Record |
| FAT | 文件分配表 | 文件系统类型 |
| POST | 上电自检 | Power-On Self-Test |
| Heap Overflow | 堆溢出 | 内存安全漏洞类型 |
| Stack Buffer Overflow | 栈缓冲区溢出 | 内存安全漏洞类型 |
| CVE | 通用漏洞披露 | 已公开漏洞的标识符 |
| Heap Sanitizer | 堆内存消毒器 | 检测堆相关漏洞的工具 |
