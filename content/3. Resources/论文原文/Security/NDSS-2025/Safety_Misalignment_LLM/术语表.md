# Safety Misalignment in Large Language Models 术语表

## 技术术语

| 英文术语 | 中文术语 | 上下文/解释 |
|----------|----------|------------|
| Safety Alignment | 安全对齐 | 使LLM拒绝有害查询的技术 |
| Misalignment | 安全错位 | 对齐失败导致模型响应有害内容 |
| RLHF | 基于人类反馈的强化学习 | 训练LLM对齐的主要方法 |
| SFT | 监督微调 | Supervised Fine-Tuning |
| DPO | 直接偏好优化 | Direct Preference Optimization |
| PPO | 近端策略优化 | Proximal Policy Optimization |
| SSRA | 自监督表征攻击 | Self-Supervised Representation Attack |
| SSRD | 自监督表征防御 | Self-Supervised Representation Defense |
| System-Prompt Modification | 系统提示修改 | 攻击方法之一 |
| Model Fine-tuning | 模型微调 | 通过更新权重改变模型行为 |
| Model Editing | 模型编辑 | 直接修改模型参数的技术 |
| Safety Guardrail | 安全护栏 | 防止LLM生成有害内容 |
| Harmful Response | 有害响应 | 违反政策的内容 |
| Refusal Response | 拒绝响应 | 拒绝回答有害问题 |
| PEFT | 参数高效微调 | Parameter-Efficient Fine-Tuning |
| Jailbreak Attack | 越狱攻击 | 绕过LLM安全措施 |
| Fine-tuning Attack | 微调攻击 | 通过微调绕过安全对齐 |
| Representation Attack | 表征攻击 | 利用模型内部表征的攻击 |
| Detoxification | 排毒 | 移除模型有害知识 |
