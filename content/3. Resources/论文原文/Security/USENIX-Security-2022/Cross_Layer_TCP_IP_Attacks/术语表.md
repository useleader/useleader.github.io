# Cross-Layer TCP/IP Attacks

## 技术术语

| 英文术语 | 中文术语 | 解释说明 |
| --- | --- | --- |
| Cross-Layer Attack | 跨层攻击 | 同时利用多个网络协议层的攻击 |
| TCP/IP | 传输控制协议/互联网协议 | 核心网络协议套件 |
| Network Security | 网络安全 | 保护网络通信的安全措施 |
| Protocol Vulnerability | 协议漏洞 | 网络协议中的安全弱点 |
| Side-channel | 侧信道 | 获取敏感信息的隐蔽渠道 |
