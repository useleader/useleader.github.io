# ABACuS 术语表

## 技术术语

| 英文术语 | 中文术语 | 解释 |
|----------|----------|------|
| RowHammer | 行锤攻击 | 重复激活DRAM行导致相邻行发生位翻转的硬件安全攻击 |
| DRAM | 动态随机存取存储器 | 计算机主内存技术 |
| Aggressor Row | 攻击行 | 频繁激活导致相邻行发生位翻转的DRAM行 |
| Victim Row | 受害行 | 被攻击行影响发生位翻转的DRAM行 |
| Row Activation Counter | 行激活计数器 | 跟踪DRAM行激活次数的硬件计数器 |
| Sibling Rows | 兄弟行 | 不同DRAM bank中具有相同行地址的行 |
| Refresh Window | 刷新窗口 | DRAM需要刷新数据的时间窗口 |
| Bitflip | 位翻转 | RowHammer攻击导致的存储数据位变化 |
| Memory Controller | 内存控制器 | 管理DRAM访问的硬件组件 |
| Bank-Level Parallelism | 银行级并行性 | 不同DRAM bank可并发访问的特性 |
| Preventative Refresh | 预防性刷新 | 在RowHammer阈值到达前刷新潜在受害行 |
| Chip Area | 芯片面积 | 硬件实现的物理面积成本 |
| RTL | 寄存器传输级 | 硬件设计描述级别 |
| ASIC | 专用集成电路 | 定制化硬件实现 |
