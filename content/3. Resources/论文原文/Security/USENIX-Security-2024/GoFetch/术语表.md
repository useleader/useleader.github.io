# GoFetch 术语表

## 技术术语

| 英文术语 | 中文术语 | 解释 |
|----------|----------|------|
| DMP | 数据内存依赖预取器 | Data Memory-Dependent Prefetcher，另一种预取器类型 |
| Constant-Time Programming | 常数时间编程 | 确保程序执行时间与秘密数据无关的编程范式 |
| Side-Channel Attack | 侧信道攻击 | 通过观察物理效应而非算法本身来窃取秘密的攻击方式 |
| Cache Side-Channel | 缓存侧信道攻击 | 利用CPU缓存状态变化获取秘密信息的攻击技术 |
| Flush+Reload | 刷新重载攻击 | 缓存侧信道攻击技术，先刷新共享缓存线再测量重载时间 |
| Prime+Probe | 先入先出探测攻击 | 缓存侧信道攻击技术，通过填充缓存集探测受害者访问 |
| Prefetcher | 预取器 | 预测并预先加载数据到缓存的硬件优化组件 |
| Eviction Set | 驱逐集 | 为特定缓存行制造冲突的一组内存地址 |
| Cryptographic Implementation | 密码学实现 | 加密算法的软件/硬件实现 |
| Key Extraction | 密钥提取 | 从密码学实现中恢复秘密密钥的攻击 |
| Constant-Time Swap | 常数时间交换 | 执行时间恒定的交换操作，用于防御侧信道攻击 |
| Pointer | 指针 | 内存地址引用 |
| Cache Line | 缓存行 | CPU缓存的基本数据单元 |
| L1/L2 Cache | 一级/二级缓存 | 处理器的多级缓存层次结构 |
| Apple M-Series | 苹果M系列芯片 | 苹果公司设计的ARM架构处理器 |
| Intel Raptor Lake | 英特尔猛禽湖 | 第13代英特尔酷睿处理器架构 |
