# Jailbreak_LLM (DRA) 术语表

## 技术术语

| 英文术语 | 中文术语 | 解释 |
|----------|----------|------|
| Jailbreak | 越狱攻击 | 绕过LLM安全机制诱导生成有害内容的攻击 |
| DRA | 伪装重建攻击 | Disguise and Reconstruction Attack，通过伪装和重建有害指令进行越狱 |
| LLM | 大语言模型 | Large Language Model，能够生成类人文本的深度学习模型 |
| Safety Fine-tuning | 安全微调 | 使用人类反馈对齐LLM安全性的训练过程 |
| Safety Alignment | 安全对齐 | 确保LLM输出符合人类价值观和伦理的过程 |
| Prompt Engineering | 提示工程 | 构造文本输入以引导AI模型输出的技术 |
| Dialogue Template | 对话模板 | 格式化用户查询和模型回复的结构化模板 |
| Payload Reconstruction | 有效载荷重建 | DRA攻击中模型重建伪装有害指令的阶段 |
| Context Manipulation | 上下文操纵 | DRA攻击中操控上下文以促进有害内容生成的技术 |
| Black-Box Attack | 黑盒攻击 | 攻击者无法访问模型内部结构和参数的攻击场景 |
| RLHF | 基于人类反馈的强化学习 | Reinforcement Learning from Human Feedback |
| SFT | 监督微调 | Supervised Fine-Tuning |
| DPO | 直接偏好优化 | Direct Preference Optimization |
| PPO | 近端策略优化 | Proximal Policy Optimization |
| Shellcode | Shell代码 | 传统软件安全中的shellcode概念启发了DRA攻击 |
| Bias | 偏差 | LLM安全微调中导致漏洞的统计偏差 |
| Self-Generated Content | 自我生成内容 | 模型在其回复中生成的有害内容 |
