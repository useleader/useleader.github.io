# SeaK 术语表

## 技术术语

| 英文术语 | 中文术语 | 解释 |
|----------|----------|------|
| Secure Allocator | 安全分配器 | 针对堆漏洞提供防护的内存分配器 |
| OS Kernel | 操作系统内核 | 管理硬件资源和提供系统服务的核心软件 |
| Heap Exploitation | 堆利用 | 利用堆内存漏洞进行攻击的技术 |
| Use-After-Free | 使用后释放漏洞 | 访问已释放内存的安全漏洞 |
| Heap Out-of-Bound | 堆越界访问 | 读写超出分配堆块边界的漏洞 |
| eBPF | 扩展伯克利包过滤器 | Extended Berkeley Packet Filter，内核虚拟机技术 |
| Atomic Alleviation | 原子缓解 | SeaK提出的选择性对象保护策略 |
| Buddy Allocator | 伙伴分配器 | Linux内核的物理页分配器 |
| SLAB/SLUB Allocator | SLAB/SLUB分配器 | Linux内核的小对象分配器 |
| Freelist Randomization | 空闲列表随机化 | 随机化空闲链表顺序的防御技术 |
| Structure Layout Randomization | 结构布局随机化 | 随机化结构体字段顺序的防御技术 |
| KFENCE | 内核模糊检查 | Kernel Fuzzing，比Guard Page更轻量的防护机制 |
| Guard Page | 保护页 | 隔离内存区域的保护机制 |
| Red Zone | 红色区域 | 对象周围用于检测溢出的安全区域 |
| Poisoning | 投毒 | 释放后填充特定值的防护技术 |
| Cross-Cache Exploitation | 跨缓存利用 | 跨越不同slab缓存的堆利用技术 |
| DirtyCred | 脏凭证攻击 | 利用内核凭证替换进行提权的攻击技术 |
| Kernel Heap | 内核堆 | 内核态的动态内存分配区域 |
| Privilege Escalation | 权限提升 | 从低权限升级到高权限的攻击 |
