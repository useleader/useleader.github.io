# G2FUZZ: LLM-based Fuzzing

## 技术术语

| 英文术语 | 中文术语 | 解释说明 |
| --- | --- | --- |
| G2FUZZ | LLM-based Graybox Fuzzing | 基于大语言模型的灰盒模糊测试 |
| LLM | 大语言模型 | 基于深度学习的大型语言模型 |
| Graybox Fuzzing | 灰盒模糊测试 | 结合白盒和黑盒方法的模糊测试 |
| Fuzzing | 模糊测试 | 通过随机输入发现程序漏洞 |
| Mutation | 变异 | 对输入数据进行随机修改 |
| Coverage | 覆盖率 | 测试覆盖的代码比例 |
