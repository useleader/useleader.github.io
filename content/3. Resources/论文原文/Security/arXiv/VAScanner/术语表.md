# VAScanner: Vulnerable API Detection

## 技术术语

| 英文术语 | 中文术语 | 解释说明 |
| --- | --- | --- |
| VAScanner | Vulnerable API Scanner | 漏洞API扫描器 |
| Vulnerable API | 漏洞API | 存在安全问题的应用程序接口 |
| Detection | 检测 | 发现和识别问题的过程 |
| API Security | API安全 | 保护应用程序接口的安全措施 |
| Static Analysis | 静态分析 | 不执行程序的分析方法 |
| Vulnerability Discovery | 漏洞发现 | 找到软件中的安全缺陷 |
