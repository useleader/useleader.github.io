# XPT: Prefetch Side Channel

## 技术术语

| 英文术语 | 中文术语 | 解释说明 |
| --- | --- | --- |
| XPT | Cross-Process Trigger Prefetch | 跨进程触发预取攻击 |
| Prefetch Side Channel | 预取侧信道 | 利用硬件预取器进行信息泄露 |
| Side-channel Attack | 侧信道攻击 | 利用系统实现细节获取敏感信息 |
| Cross-Process | 跨进程 | 涉及不同进程的攻击 |
| Cache Attack | 缓存攻击 | 通过缓存时序差异获取信息 |
| Hardware Prefetcher | 硬件预取器 | 预测并预加载数据的硬件组件 |
