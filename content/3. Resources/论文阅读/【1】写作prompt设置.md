## 🧩 一、总体思路

你要让模型经历三个阶段：

1. **学习阶段（In-context Learning）**：让模型深入阅读10篇论文，抽取期刊的共同风格与逻辑特征；
    
2. **归纳阶段（Meta-summarization）**：让模型总结该期刊的写作规律；
    
3. **应用阶段（Guided Writing）**：让模型基于这些规律指导你写自己的论文段落。
    

---

## 🧠 二、核心提示词模板

### （1）阶段一：学习阶段 — 摘取风格特征

> 角色设定 + 明确任务目标 + 提取维度

**提示词：**

```
你现在是一个专门研究学术期刊写作规律的AI学术编辑。
我将给你输入10篇来自同一期刊的论文文本（或摘要、引言、结论部分）。

请你仔细阅读这些文本，从以下角度提炼规律：
1. 研究主题和方法学倾向（理论/实证/实验/模型/调查）
2. 摘要的组织方式（背景—目的—方法—结果—结论的比例）
3. 引言中引用文献的逻辑（是否强调前人研究不足？是否逐层收窄？）
4. 段落衔接方式与句式特征（例如使用连接词、过渡短语）
5. 常用表达模版（如 “In this study, we propose …”, “The results demonstrate that …”）
6. 语气与态度（是否偏客观中性、是否喜欢强调创新性或实际意义）
7. 结论与未来工作写作模式

请输出一个结构化总结（例如以表格或分点方式呈现），并提炼出该期刊独有的“写作风格指南”。
```

> ✅ **技巧**：
> 
> - 如果论文太多，先输入摘要和引言部分即可。
>     
> - 可以一次输入2-3篇，用提示词让模型“持续积累学习”，比如：
>     
>     ```
>     请继续学习以下论文文本，并在你已有总结基础上更新规律。
>     ```
>

### （2）阶段二：归纳阶段 — 输出期刊风格规则

在模型已经读完论文后，用以下提示让它输出“写作风格画像”：

**提示词：**

```
基于你阅读的10篇论文，请总结该期刊论文的整体写作逻辑和语言风格规律。
请用以下结构输出：

【期刊风格画像】
- 研究主题与学科偏好：
- 典型论文结构模式：
- 常见逻辑推进方式：
- 摘要写作风格：
- 引言组织逻辑：
- 语言语气与表达特征：
- 典型句式与常用短语：
- 写作禁忌或不常见表达：

【简要总结】
用100字以内总结该期刊论文最典型的写作特征（如“一种稳健、逻辑严密且重视创新价值的科技论文风格”）。
```

> ✅ 这样你得到的是一份可复用的“风格指令表”，后续写作时可直接套用。

---

### （3）阶段三：应用阶段 — 写作指导提示词

现在你要让模型充当一个“期刊风格顾问”，结合你自己的论文主题进行指导。

**提示词：**

```
你现在是一名精通[期刊名]写作风格的AI学术编辑。
以下是该期刊的写作规律摘要（粘贴上阶段的总结内容）。

请根据这些规律，指导我撰写一篇关于“XXXX”的论文。
我需要你帮我完成以下任务：
1. 给出符合该期刊风格的论文大纲；
2. 帮我改写下面的摘要，使其符合该期刊的语气和逻辑；
3. 给出10个符合该期刊风格的常用句式模板，用于摘要和引言。

（如果我输入论文段落，请帮我评估是否符合该期刊风格，并指出可改进的地方。）
```

## 🚀 四、可选强化技巧

- **用“多轮记忆法”构建风格画像**：  
    把每3篇论文学习后的总结都让模型保存为“已学习内容”，  
    最后让它“整合前面所有学习成果”生成最终规则。
    
- **多模型对比法**：  
    可以让 DeepSeek、ChatGPT、Claude 等分别分析，综合其共识规律。
    
- **输出样例强化**：  
    让模型举出3个“风格化的句子样例”→ 帮助你掌握句式节奏。
