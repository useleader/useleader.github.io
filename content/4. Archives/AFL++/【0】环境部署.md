## Building and installing AFL++
镜像重命名
```bash
docker tag docker.io/aflplusplus/aflplusplus:latest aflplusplus:latest
```

创建启动一个可交互的容器命令行终端，并创建数据卷挂在，将本地文件映射到/src目录，同时实现数据共享
```shell
docker run -ti -v /location/of/your/target:/src aflplusplus/aflplusplus
```

详细的教程：[[【0.4】 参考资料 - Building and installing AFL++]]
